Welcome to the BRUT.E Podcast digital showcase! This project was designed and developed under the "Vibe Coded" philosophy: prioritizing deep aesthetics, raw emotion, and immersive cinametric web design over generic templates.

Note: The source code for this project is private and proprietary to protect its unique design architecture. This repository serves as a visual showcase of the frontend development and UI/UX design.

### 🎨 The Aesthetic Vision
BRUT.E is hosted by Leya and talks about being in your 20s, raw emotions, vulnerability, life struggles, friendships, and mental health.

The website was built to feel:

RAW & INTIMATE: Like a digital diary mixed with a cultural magazine.
HONEST & MODERN: Unfiltered, bold, and editorial.
Visual Identity
Palette: Warm light beige (off-white) background, deep podcast red (#9E0F16), soft shadow red, and dark grey.
Typography:
Primary: Bold, condensed, massive sans-serif (Anton) for maximum editorial impact.
Secondary: Clean, modern sans-serif (Inter) for readability.
Accents: Handwritten style (Caveat) for the logo and authentic diary-like notes.
Textures: Subtle paper grain overlay applied via CSS to give it a physical magazine feel.

### ⚡ Core Technical Features (Vibe Coded)
This project was built from scratch without standard UI libraries (No Tailwind, No Bootstrap) to ensure 100% bespoke pixel control.

1. The Immersive Preloader
A full-screen cinematic entry sequence powered by GSAP. An animated retro phone cord winds across the screen before the handwritten logo fades in, dissolving smoothly into the main site.

2. The 3D "Retro Phone" Floating CTA
A custom-built, fixed floating element living in the bottom right corner. Instead of using complex WebGL, the 3D rounded depth is achieved purely through advanced CSS box-shadow layering and transform hover states.

3. Asymmetric Editorial Grid Layouts
Dynamic Marquees: Massive typography overlapping the viewport.
Library Views: A custom toggle allowing users to switch the Spotify Episode library between a minimalist Vertical List and a highly stylized Visual Cover Grid.
4. Smooth Cinematic Motion
Every page transition, text reveal, and hover effect is carefully orchestrated using GSAP (GreenSock Animation Platform) and Next.js routing transitions.

## 🛠️ Technology Stack
Framework: Next.js (App Router)
Language: TypeScript
Styling: Pure Modular Vanilla CSS (for ultimate customization)
Animation Engine: GSAP (@gsap/react)
Iconography: Lucide React
Integrations: Spotify Embed APIs
